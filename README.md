# 深夜ローソン・サバイバルファイター

**SHINJUKU LOWSON SURVIVAL FIGHTER** — 深夜の歌舞伎町コンビニで「殴って勝つ」のではなく **避けて・記録して・説得して・みんなで対応する** 1画面 SFC 風 2D 格闘ゲーム風プロトタイプ。

## 起動

`index.html` をブラウザにドラッグ＆ドロップするだけ（あるいはダブルクリック）。
ローカルサーバー、`npm`、コンパイル、追加ファイル — **何も不要**。
画像は HTML に base64 で埋め込み済み（約 6MB の単一ファイル）。

## 操作（ワンボタン化）

| キー | 役割 |
|---|---|
| ← / → | 移動 |
| ↓ | しゃがみ（カバン振り・空き缶を回避） |
| Z | ガード／（+ ←/→ でバックステップ） |
| **X 長押し** | **距離で自動切替** — 近=**説得** / 遠=**撮影** |
| Space | 突き返し（反撃／空き缶パリィ／つかみ脱出） |
| Enter | 決定 |
| H | タイトル画面で詳細ヘルプ |

**行動はぜんぶ X 一個**。敵との距離で説得 / 撮影 が自動で切り替わります。
床に **緑帯**（説得圏）と **青帯**（撮影圏）が常時表示されるので、自分の立ち位置を見るだけでどのモードに入れるかわかります。

## ゲームの流れ

### 説得ゲージ（緑） — 近距離で X 長押し
- 敵の半径 90px 以内で蓄積。同時に敵の迷惑度が下がる
- **50%** で **先輩** が自動で出てきてくれる（バックヤードから）
- **100%** で **店長** も自動で出てくる
- 近いから振り回し・つかみ・怒鳴りに注意

### 撮影ゲージ（青） — 中遠距離で X 長押し
- 敵から 90〜240px の距離で蓄積。これが **自動通報** の素
- **50%** で **警備会社** が自動到着
- **100%** で **警察** も到着 → ROUND 2 突入
- 距離があるので比較的安全（ただし空き缶が飛んでくる）

### ROUND 2 — みんなで店外退去
- 集まった応援が敵に密着して**押し出す** → EVICT ゲージ蓄積
- 協力ボーナス: 1 人=1.0× / 2 人=1.4× / 3 人=1.7× / **4 人=2.0×**
- → 説得・撮影 両方やって 4 人態勢にできれば爆速で押し出せる

### サポートはやられる
- 敵の振り回し・空き缶投げ・怒鳴り近接で削れる
- スタミナ 0 で **左へ撤退**（KO 表示でグレーアウト → 画面外で消える）
- 役割ごとの撤退セリフ：
  - 先輩: 「俺もう無理…」「シフト終わりだから帰るわ…」
  - 店長: 「本部に確認します…」「ユウマくん…後は頼んだ」
  - 警備: 「契約範囲外なんで…」「外で待機しときますね…」
  - 警察: 「民事不介入なんで…」「上司に確認します…」
- 撤退者は時間経過で**自動再来店**（先輩/店長は短め、警備/警察は長め）

### 敗北
- ストレス満タン（**時間制限はナシ**。じっくり整えてOK）
